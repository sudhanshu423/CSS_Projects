# 🛒 Project 1 – Amazon Clone

This project is a front-end clone of the Amazon website, created to practice modern web development skills. It replicates key parts of Amazon's user interface, including the homepage, product listings, cart functionality, and responsive design. It's built using HTML, CSS, and JavaScript (optionally with backend tools like Firebase or Node.js).

## 📌 Description

- The Amazon Clone is a beginner-to-intermediate level project that demonstrates how to build a simplified version of an e-commerce platform. While it doesn't include all of Amazon's advanced features, it showcases the core structure and functionality required for online shopping platforms.

- This project helped me understand layout techniques, interactivity with JavaScript, and how to simulate a dynamic user experience.

## ✨ Features

- 🖥️ Homepage with Product Listings

- 🔍 Clickable Product Cards

- 🌙 Clean UI Styled with CSS / Flexbox / Grid
  
- 📁 Organized file structure and modular code

# 🧠 What I Learned

Through this project, I learned and practiced:

- ✅ Structuring HTML pages for large-scale websites

- 🎨 Styling complex layouts using CSS Flexbox and CSS Grid

- 🧠 Understanding the difference between block, inline, and inline-block


🛍️ Managing state like cart items (with JavaScript or LocalStorage)

🌐 Building responsive web pages for multiple screen sizes

🔐 (Optional) Integrating Firebase Authentication

💳 (Optional) Implementing Stripe for secure payments

🚀 (Optional) Deploying a project using GitHub Pages, Netlify, or Vercel
